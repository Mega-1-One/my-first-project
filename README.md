# \# Weather Dashboard

# 

# A simple weather dashboard that fetches real-time weather data from OpenWeatherMap API.

# 

# \## Features

# \- Search weather by city name

# \- Display current temperature, weather conditions, humidity, and wind speed

# \- Beautiful gradient UI

# \- Responsive design

# \- Real-time data from OpenWeatherMap API

# 

# \## How to Use

# 1\. Open `index.html` in your web browser

# 2\. Enter a city name in the search box

# 3\. Click "Search" or press Enter

# 4\. View the current weather information

# 

# \## Technologies Used

# \- HTML5

# \- CSS3 (with gradients and flexbox)

# \- JavaScript (async/await, Fetch API)

# \- OpenWeatherMap API

# 

# \## API

# This project uses the free OpenWeatherMap API: https://openweathermap.org/api
